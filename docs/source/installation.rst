============
Installation
============

Dependencies
------------

`galport` requires the following Python packages:

* `numpy <https://numpy.org/>`_
* `scipy <https://scipy.org/>`_
* `agama <https://github.com/GalacticDynamics-Oxford/Agama>`_ 
* `naif <https://github.com/lberaldoesilva/naif>`_ (optional)

Installing
----------

`galport` can be installed from the PyPI module typing:

.. code-block:: bash
   
   pip install --user galport

You can also install the latest development version directly from GitHub:

.. code-block:: bash

   git clone https://github.com/vdzozulia/galport.git
   cd galport
   pip install -e .
